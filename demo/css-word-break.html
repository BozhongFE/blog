一、强制不换行

p {
  white-space: nowrap;
}

1. white-space: nowrap 强制在同一行内显示所有文本，直到文本结束或者 br 标签。 

p {
  white-space: pre;
}

2. white-space: pre 换行和其他空白字符(换行符/空格/制表符/文字转行)都将受到保护,等同 pre 标签。 

p {
  word-break: keep-all;
}

3. word-break: keep-all 不允许CJK（中/日/韩）文本断行，移动端基本不支持。例如这里测试的这一句

二、强制换行

pre {
  white-space: pre-wrap;
}
    pre 换行：
    a. 可用 white-space: pre-wrap 但过长的英文/数字会另起一行，特殊情况下会溢出空间，而且不同浏览器对空白换行的处理有差异。
        
pre {
  white-space: pre-line;
}
    b. 也可以用white-space: pre-line，但会合并空格和制表符。
        
p {
  word-wrap: break-word;
}

英文数字内换行：
word-wrap: break-word 会先尝试将单词挪到下一行，下一行宽度不够才会进行单词内断句，例如： aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa 

p {
  word-break: break-all;
}

word-break: break-all 直接进行单词内断句，不会将单词挪到下一行。 aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa 

p {
  white-space: nowrap;
}

display: block 块级元素文字溢出不会撑起容器。示例默认都是块级 

p {
  width:auto;
  display: inline-block
  white-space: nowrap;
}

display: inline-block 行内块元素若不定宽度，文字溢出会撑起容器。word-break:break-all 有效。word-wrap: break-word 需配合宽度使用(不同浏览器有差异) 

p {
  width:auto;
  display: inline-block;
  word-wrap: break-word;
}

word-break:break-all 有效。word-wrap: break-word 需配合宽度使用 ssssssssssssssssssssssssssssssssssssssssssssssssssssssssssssssssssssssssss (不同浏览器有差异)

  word-break:break-all;
